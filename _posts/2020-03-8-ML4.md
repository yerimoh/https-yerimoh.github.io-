# SVM이란?
서포트 벡터 머신 SVM(support vector machine)  
* 매우 강력함: 머신러닝 모델 중 가장 인기있는 모델 중 하나임         
* 다목적 머신러닝 모델: 선형 or 비선형 분류 회귀, 이상치 탐색에 사용가능     
* 사용 용도    
   * 특히 복잡한 분류 문제에 적합      
   * 작거나 중간 크기의 데이터셋에 적합
