## 5) 간단한 출력으로 작동 시험  

작동 시험을 위해 'hello world' 출력   
(출력을 위한 문법은 나중에 차차 설명합니다)

**실행은 [F11]**

```java
public class a {

	public static void main(String[] args) {
	 	// TODO Auto-generated method stub
  
        // 아래 한 줄만 복사해 보세요!
	System.out.println("hello world");
	}
```
